# Sistema de Gestión de Tareas (Mini-Core UDLA)

![.NET 10](https://img.shields.io/badge/.NET-10.0-blue)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-336791?logo=postgresql&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-2CA5E0?logo=docker&logoColor=white)
![Render](https://img.shields.io/badge/Render-46E3B7?logo=render&logoColor=white)

Aplicación MVC para la gestión de avances con lógica de filtrado por fechas y cálculo de retrasos laborables (excluyendo fines de semana).

**Demo Link:** [https://tarea4-ejerciciomvc-andrade.onrender.com](https://tarea4-ejerciciomvc-andrade.onrender.com)

## Tecnologías Utilizadas
* **ASP.NET MVC** (Base del framework web y arquitectura).
* **Entity Framework Core** (ORM y migraciones).
* **PostgreSQL** (Persistencia y proveedor de datos relacional).
* **DataTables y Bootstrap 5** (Interfaz de tablas, paginación y diseño de filtros interactivos).
